# JARVIS Lite — планер дня

Личный планер дня в браузере. **Бесплатный, без сервера, без API-ключей.**
Все данные хранятся только на вашем устройстве (IndexedDB). Ставится на телефон как PWA.

## Что умеет

- Вводите задачи текстом (по строке на задачу) — приложение раскладывает их по категориям,
  приоритетам и времени по ключевым словам.
- Строит расписание дня с перерывами, обедом и свободными окнами.
- В свободные окна предлагает микро-обучение по вашим интересам (факты, слова, этимология,
  шахматные задачи, кино) из встроенного набора карточек.
- Напоминает о старых невыполненных задачах (плашка «ждут со вчера»).
- Вечером подводит итог дня и переносит невыполненное на завтра.
- Честная статистика: выполнено по дням, процент, серия дней, категории, время в фокусе.
  Никаких выдуманных «продуктивность выросла на X%».

## Запуск

Нужен Node.js 18+ ([nodejs.org](https://nodejs.org) — поставьте LTS-версию).

```bash
npm install     # один раз
npm run dev     # запустить локально → откроется http://localhost:5173
```

## Сборка и бесплатный хостинг

```bash
npm run build   # готовая статика окажется в папке dist/
npm run preview # проверить собранную версию локально
```

Папку `dist/` можно бесплатно выложить на GitHub Pages, Netlify или Vercel — сервер не нужен.

## Проверить, что всё работает

1. **Добавить** → введите несколько строк, например:
   `доделать отчёт по работе это важно`, `сходить в зал в 18:00`, `позвонить маме если успею`.
   Нажмите «Разобрать» → «Построить день».
2. **Сегодня** → появится расписание. Отмечайте задачи галочкой.
3. В свободном окне — карточка с фактом/словом; «Ещё» даёт новую.
4. **Вечер** → «Перенести на завтра» у невыполненной задачи; наутро она в плашке «ждут со вчера».
5. **Статистика** → раскройте «Для теста» → «Заполнить тестовыми данными», чтобы увидеть графики за неделю.

## Как работает распознавание

Без искусственного интеллекта — простыми правилами в `src/logic/` и словарём в
`src/logic/keywords.ts`. Категории, приоритеты и время угадываются по ключевым словам;
после «Разобрать» всё можно поправить руками. Словари легко дополнять своими словами.

## Ограничения (осознанные, для MVP)

- Уведомления работают только пока приложение/вкладка открыты (нет push-сервера).
- Нет синхронизации между устройствами и аккаунтов — данные живут в одном браузере.
- Распознавание проще, чем у «умной» версии: вводите задачи понятными фразами.

## Апгрейд до умной версии

Четыре «умных» функции (`src/logic/parseTasks.ts`, `buildDay.ts`, `daySummary.ts` и
`src/content/suggestions.ts`) написаны с чистыми сигнатурами. Когда захотите живой разбор
сбивчивой речи и генерируемые факты — их можно точечно заменить на вызовы Claude API.
Готовые промпты и JSON-схемы для этого лежат в отдельной полной спецификации.
